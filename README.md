💍 Wedding Invitation Website

A beautifully designed digital wedding invitation website to share our special day with friends and family. This project presents event details in an elegant and interactive way.

✨ Features

Elegant hero section with couple names

“Save the Date” 

Image gallery for 

Custom fonts and romantic design 

Fully responsive layout (mobile-friendly)

Smooth scrolling and simple animations

Built With

HTML5

CSS3

Tailwind CSS

Custom fonts & assets

📁 Project Structure

Wedding-website-main/
│── index.html
│── style.css
│── hero.jpg
│── images/
│── fonts/

💡 Customization

You can easily personalize this template by:

Changing names in index.html
Replacing images in the /images folder
Updating event details (date, venue, etc.)
Editing styles in style.css
